Esta aplicación está en proceso de desarrollo

Será una herramienta desarrollada para llevar el registro de las ventas diarias que realiza una cooperativa de 10 padres de la escuela.

De manera pública se puede acceder los datos de los emprendedores, productos y precios disponibles (por ahora todo es público)

De manera privada, luego del login, se pueden grabar las ventas realizadas por el usuario y obtener un informe de ventas del día. También publicar, editar o eliminar productos del usuario logueado.

Sólo pueden loguearse usuarios autorizados por el administrador del sistema.
