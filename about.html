Esta aplicación está en proceso de desarrollo.

Será una herramienta creada para llevar el registro de las ventas diarias que realiza una cooperativa de 10 padres de una escuela.

De manera pública se puede acceder a los datos de los emprendedores, productos y precios disponibles (por ahora todo es público)

De manera privada, se pueden grabar las ventas realizadas por el usuario logueado y obtener un informe de las ventas totales del día. También se podrá publicar, editar o eliminar productos del usuario logueado.

Los usuarios que pueden loguearse serán autorizados por el administrador del sistema.
